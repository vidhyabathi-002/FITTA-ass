# Simple React Calculator

A simple calculator built with React that adds two numbers entered by the user.

## Features

- Input two numbers
- Add them together
- Display the result

## Getting Started

### Prerequisites

- Node.js version 20.19+ or 22.12+ (due to Vite requirements)
- npm

### Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server

### Usage

Enter two numbers in the input fields and click "Add" to see the sum.
