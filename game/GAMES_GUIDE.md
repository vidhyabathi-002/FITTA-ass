# 🎮 Game Collection - Quick Start Guide

This folder contains multiple pygame games! Here's what's available:

## 🚀 Games Available

### 1. **Collision Dodger** - `main.py`
Simple beginner-friendly game where you dodge falling squares.
```powershell
python main.py
```
- Move with arrow keys
- Avoid red squares
- Track your score

---

### 2. **Space Shooter** - `space_shooter.py`
Classic space invaders-style game with waves and difficulty progression.
```powershell
python space_shooter.py
```
- Shoot enemies with SPACE
- Survive multiple waves
- Collect power-ups
- **Features:** Enemy bullets, wave system, lives

---

### 3. **Space Shooter ULTIMATE** - `space_shooter_ultimate.py` ⭐ **BEST**
Fully featured arcade shooter with ALL features!
```powershell
python space_shooter_ultimate.py
```
- **4 different weapon types** - Single, Dual, Triple, Laser
- **Boss enemies** - Every 5th wave
- **Particle explosions** - Beautiful destruction effects
- **Sound effects** - Shooting, explosions, power-ups
- **Health system** - Track player and boss health
- **Power-ups** - Health, Weapons, Shields
- **Dynamic difficulty** - Gets harder each wave

---

## 🚀 RECOMMENDED STARTING POINT

## ⚙️ SETUP INSTRUCTIONS

1. **Activate virtual environment:**
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

2. **Install dependencies:**
   ```powershell
   pip install -r requirements.txt
   ```

3. **Run any game:**
   ```powershell
   python [game_name].py
   ```

---

## 🎮 UNIVERSAL CONTROLS

| Key | Action |
|-----|--------|
| **← →** | Move left/right |
| **↑ ↓** | Move up/down |
| **SPACE** | Shoot/Action |
| **R** | Restart (when game over) |
| **Q** | Quit (when game over) |
