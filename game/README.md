# Simple Pygame Game

A basic collision avoidance game built with Pygame.

## Game Rules
- Use **Arrow Keys** to move the green square
- Avoid the red falling squares
- Each square you dodge increases your score
- The game ends if you hit a red square

## How to Run

1. **Activate the virtual environment:**
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

2. **Run the game:**
   ```powershell
   python main.py
   ```

## Controls
- **← →** : Move left/right
- **↑ ↓** : Move up/down
- **Close window** : Exit game

## Dependencies
- pygame

## To Install Dependencies
```powershell
.\venv\Scripts\pip install -r requirements.txt
```
